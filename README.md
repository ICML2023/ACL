## Simple and Asymmetric Contrastive Learning for Graphs


### Requirements

- Python version: 3.7.10
- Pytorch version: 1.10.2
- DGL version: 0.7.1
- networkx
- scipy
- seaborn
- tqdm




### Experiments
- To run node classification on homophily graphs 
```
sh run_homo/run.sh
```


- To run node classification on heterophily graphs 
```
sh run_heter/run.sh
```
